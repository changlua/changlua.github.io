腾讯云域名+Github绑定

前提准备：你的github仓库地址 如changlua.github.io

1、购买域名，实名认证

对于域名的购买我们可以选择腾讯云

腾讯云地址：https://cloud.tencent.com/

没有注册的先注册一下，并且进行实名认证(认证还是很快的)


如何查看自己购买的域名？


以下则为我接下来演示的域名：

2、增加解析地址


点击解析：

点击添加记录：


  添加两条，两条的主机记录与记录类型与我一样
  第一条的记录值填写对应的github网址的ip地址(可以通过cmd，输入指令ping github地址获取)


  第二条的记录值填写你自己的github博客地址


3、在GitHub中Custom domain添加购买的域名

进入仓库，点击旁边的setting：


继续往下翻找到Custom domain，填入你购买的域名地址：


填入之后你的Github仓库中就会生成一个CNAME

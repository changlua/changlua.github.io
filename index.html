腾讯云域名绑定Github

发表于 2020-08-21

腾讯云域名+Github绑定

前提准备：你的github仓库地址 如changlua.github.io

1、购买域名，实名认证

对于域名的购买我们可以选择腾讯云

腾讯云地址：https://cloud.tencent.com/
没有注册的先注册一下，并且进行实名认证(认证还是很快的)


如何查看自己购买的域名？


以下则为我接下来演示的域名： 建议是最好实名注册一下



2、增加解析地址


点击解析：

点击添加记录：
  添加两条，两条的主机记录与记录类型与我一样
  第一条的记录值填写对应的github网址的ip地址(可以通过cmd，输入指令ping github地址获取)
  第二条的记录值填写你自己的github博客地址


3、在GitHub中Custom domain添加购买的域名(可跳过看4)

进入仓库，点击旁边的setting：


继续往下翻找到Custom domain，填入你购买的域名地址：


填入之后你的Github仓库中就会生成一个CNAME


4、博客目录/source中添加CNAME，并填入注册域名


进入到你的本地博客目录中：新建CNAME文件，其中添加你的注册的域名

说明：第3步与第四步区别在于第四步在每次部署的时候都会将CNAME上传，如果只在第三步进行设置只是临时设置，下次重新部署还是不存在CNAME。



5、重新进行部署进行访问

在该文件目录中输入命令：
hexo clean && hexo g && hexo d

如下:

部署成功后即可访问你购买的域名了！
